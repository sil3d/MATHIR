## 🚀 Quick Use

### Step 1: Install Ollama

```bash
# Windows: Download from https://ollama.ai/download

# Linux
curl -fsSL https://ollama.ai/install.sh | sh

# macOS
brew install ollama
```

### Step 2: Download a Model

```bash
# If you have 8GB+ VRAM (RTX 3060/4060+)
ollama pull llama3.1:8b

# Otherwise (or for CPU only)
ollama pull llama3.2:3b
```

```bash
# Option 1: Pure Python
```

## 🐛 Quick Troubleshooting

### Ollama not detected

```bash

ollama --version

# Restart terminal
# Rerun benchmark
```

### No Ollama tab

```
Possible causes:
1. Ollama not installed
2. Model not downloaded
3. Benchmark run without Ollama

Solution: Rerun after installation
```

### Ollama timeout

