5 Why It Works: Theoretical Foundation

Episodic Memory Works Because

Cosine similarity on stored embeddings provides real recall improvement. The brain's episodic memory similarly stores experiences for faster future retrieval. FIFO eviction ensures bounded memory with predictable worst-case latency.

Immunological Memory Works Because

Mahalanobis distance is the NP-optimal detector for anomalies in Gaussian-distributed data. Like the immune system, it learns "self" (normal distribution) and detects "non-self" (anomalies). Exponential moving average adapts to concept drift without catastrophic forgetting.

Working Memory Works Because

Multi-head attention on recent context produces context-dependent query vectors. Like human working memory, recent context influences interpretation of new information. Circular buffer provides bounded latency regardless of context history length.

KL Router Works Because

KL divergence constraint prevents any single tier from dominating (gradient penalty style). The router learns to route based on query structure, not hard-coded rules. Maximum entropy objective ensures exploration of all tiers.
